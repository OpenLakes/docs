# OpenLakes Harbor

OpenLakes Harbor is the managed version of the OpenLakes platform. We handle infrastructure, configuration, and operations so you can focus on building data pipelines.

## What is Harbor?

Harbor provides all the same components as OpenLakes Core—Spark, Trino, Airflow, Superset, JupyterHub, OpenMetadata, and more—fully integrated and managed for you. No Kubernetes expertise required.

## Trial Features

The 30-day free trial includes:

- **All 20+ integrated components** - Spark, Trino, Airflow, Superset, JupyterHub, Meltano, OpenMetadata, and more
- **Single sign-on** - One login for all services
- **Pre-configured integrations** - Storage, credentials, and networking already wired
- **Example notebooks and pipelines** - Sample data workflows to help you get started
- **Web-based access** - No local installation required

## Harbor Pro (Coming Q1 2026)
